# SAA-C03 Group Map

Minimal Next.js page that shows AWS service groups in scope for SAA-C03.

## Run

1. npm install
2. npm run dev
3. Open http://localhost:3000

## Source references used

- SAA-C03 in-scope services list:
  https://docs.aws.amazon.com/aws-certification/latest/examguides/saa-03-in-scope-services.html
- AWS architecture icon package (for category colors):
  https://aws.amazon.com/architecture/icons/
